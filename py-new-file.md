# Automatische Registry-Erweiterung für Python-Dateien (Python 3.14.2)

Diese Anleitung beschreibt, wie man unter Windows:

- Neue Python-Dateien mit einem Template über das **„Neu“-Kontextmenü** erstellt.
- Rechtsklick-Menüeinträge für **Python 3.14.2** hinzufügt.


---

## 1. Ziel

- Rechtsklick → Neu → Python-Datei erstellen.
- Automatisches Template:

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-

def main():
    pass

if __name__ == "__main__":
    main()
